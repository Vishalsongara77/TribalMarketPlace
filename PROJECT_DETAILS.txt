Project Description

Tribal Marketplace is an online e-commerce platform designed to empower tribal artisans by connecting them directly with global customers. The platform facilitates the sale of authentic handcrafted products including jewelry, textiles, pottery, woodwork, metalwork, paintings, and sculptures.

Key Features

For Buyers
- Browse and search tribal products by category
- Add items to cart and wishlist
- Secure checkout with multiple payment options
- Order tracking and order history
- Product reviews and ratings
- Recommended products section
- Buyer Dashboard with quick actions

For Sellers/Artisans
- Product management and inventory tracking
- Order management system
- Real-time analytics and earnings tracking
- Seller/Artisan Dashboard
- Business metrics and performance tracking
- Quick actions for product management

For Admins
- Comprehensive dashboard with key metrics
- User management and verification
- Product approval system
- Order management and tracking
- Analytics and reporting
- Pending approvals management
- Artisan verification system

General Features
- User authentication and authorization
- Role-based access control (Admin, Seller, Artisan, Buyer)
- Real-time order status updates
- Product categorization and filtering
- Shopping cart functionality
- Multiple payment methods (COD, UPI, Card, Google Pay, PhonePe, Paytm, PayPal)
- Contact form and customer support
- Responsive design with Tailwind CSS
- Secure API with CSRF protection and rate limiting

Technology Stack

Backend
- Framework: Express.js (Node.js)
- Database: MongoDB Atlas
- Authentication: JWT (JSON Web Tokens)
- Security: Helmet, CORS, Express Rate Limit, CSRF Protection
- File Upload: Cloudinary, Multer
- Email: Nodemailer
- Real-time Communication: Socket.IO
- Payment: Razorpay integration
- Validation: Express Validator

Frontend
- Framework: React 18.2
- Build Tool: Vite
- Styling: Tailwind CSS
- Component Animation: Framer Motion
- Icons: React Icons, Lucide React
- HTTP Client: Axios
- Form Management: React Hook Form
- Notifications: React Hot Toast
- Image Gallery: React Image Gallery
- Routing: React Router v6

Development Tools
- Version Control: Git
- Code Quality: ESLint
- Task Runner: Concurrently (for concurrent development)
- Server Reloading: Nodemon

Database Models (summary)

User
- Name, email, hashed password
- Phone, address
- Role (buyer, seller, artisan, admin)
- Profile info, tokens, seller info, timestamps

Product
- Name, description, price
- Category, seller reference, images, stock
- Rating, reviews, active status, specs, tags, timestamps

Order
- Order number, buyer reference, items array
- Pricing breakdown, shipping/billing, payment status
- Order status history, tracking, timestamps

Review
- Product and user refs, rating, title, comment
- Images, helpful votes, verified purchase, timestamps

Category
- Name, slug, description, parent, image, SEO, timestamps

Cart
- User ref, items array, totals, timestamps

Coupon
- Code, type, value, limits, applicable products, expiration

Chat/Message
- Participants, product ref, messages, attachments, timestamps

Authentication & Authorization

- JWT-based auth, tokens in localStorage
- Role-based access control
- CSRF protection, bcryptjs for hashing
- Auto-logout on token expiration

UI/UX

- Responsive design, Tailwind CSS
- Gradient color scheme, smooth animations (Framer Motion)
- Toast notifications, modal dialogs, loading states
- Pagination, search & filters, progress bars

Security

- HTTPS-ready env variables
- CORS, Rate limiting (100 requests per 15 minutes)
- CSRF tokens, Helmet.js, input validation/sanitation
- Secure JWT secret management

Deployment

- Backend: Node.js-ready (Heroku, Railway, Render)
- Frontend: Vite optimized build
- Database: MongoDB Atlas
- File Storage: Cloudinary
- Environment-based config, Nodemon for dev

Getting Started (quick)

Install dependencies
npm install

Development (running frontend + backend concurrently)
npm run dev

Backend only
npm run server

Frontend only
npm run client

Production build
npm run build

Start production server
npm start
